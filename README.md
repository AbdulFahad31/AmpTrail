# 🔋 AmpTrail — EV Charging Slot Booking Platform

AmpTrail is a modern EV charging slot booking application developed using Flutter and Firebase, designed to simplify the process of locating and reserving EV charging stations in real time.

The platform enables users to discover nearby charging stations, check slot availability, book charging sessions, and receive instant updates regarding booking status through a seamless mobile experience.

---

# 📱 Features

## 👤 User Features

- 🗺️ Interactive Mapbox integration to locate nearby charging stations
- 📍 Real-time station details including pricing, availability, and ratings
- 📅 Slot booking with date, time, and duration selection
- 🔔 Instant notifications for booking approvals or rejections
- 📜 Booking history management for upcoming and past reservations
- 👤 User profile and account management

---

## 🛠️ Admin Features

- 📨 Manage incoming booking requests
- ✅ Accept or reject bookings with status updates
- 🔔 Send real-time booking notifications to users
- 🔐 Role-based admin access and management

---

# 🔐 Authentication & Security

- 📱 OTP-based phone number authentication using Firebase Authentication
- 🔐 Role-based routing for users and admins
- ☁️ Real-time data synchronization with Cloud Firestore
